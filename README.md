# Number Guessing Game 🎮

A command-line based number guessing game built using Python. The game includes multiple difficulty levels, limited attempts, and real-time feedback for user guesses.

## 🚀 Features

- Difficulty levels:
  - Easy (1–50, 10 attempts)
  - Medium (1–100, 8 attempts)
  - Hard (1–200, 6 attempts)
- Limited attempts based on difficulty
- Real-time hints (Too High / Too Low)
- Input validation using try-except
- Exit option to quit the game anytime
- Displays remaining attempts after each guess

## 🛠️ Technologies Used

- Python
- Random module

## ▶️ How to Run

1. Make sure Python is installed
2. Run the program:
   ```bash
   python game.py
